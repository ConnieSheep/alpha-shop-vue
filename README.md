# alpha-shop-vue
* 使用 Vue.js 框架打造而成的購物車系統

## 打開終端機，使用"cd + 資料夾名稱" 進入想放置此檔案的資料夾，再輸入以下：
```
git clone https://github.com/ConnieSheep/alpha-shop-vue
```
## 進入檔案資料夾：
```
cd alpha-shop-vue
```
## 環境設置：
```
npm install
```

## 執行檔案：
```
npm run serve
```

## 若出現以下畫面，代表執行成功，在瀏覽器輸入"Local"的網址即可
```
App running at:
  - Local: 'url'
  - Network: 'url'
```

## 若是要退出執行，請在終端機按下：
```
CTRL + C
```
